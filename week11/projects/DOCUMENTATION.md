# Automate creating project folder
This simple Bash script automates the creation of a new project directory, generates two default files (`README.md` and a custom file), and initializes a Git repository inside it.

## Features
* Creates a custom directory.
* Automatically creates `README.md` and a custom file.
* Initializes a Git repository inside the new directory.
* Displays a custom ASCII art logo upon execution.

## Usage

Run the script by providing a **folder name** and a **file name** as arguments:

```bash
./createProject <folder_name> <file_name>
```
