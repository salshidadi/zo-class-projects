# Student Data Structure

A C program that implements a student management system using data structures and algorithms.

# Overview

The program stores and manages student records, supporting the four fundamental data operations: insert, delete, update, and retrieve.

# Concept

The project focuses on:

- Data structures in C
- Dynamic memory management
- Implementing core data operations
- Managing structured records

Each student record holds relevant data (such as name, ID, and grade). The program provides operations to manage these records efficiently using an appropriate data structure.

# How It Works

1. The program initializes a data structure to hold student records
2. The user performs one of the supported operations:
   - **Insert**: Add a new student record
   - **Delete**: Remove an existing student record
   - **Update**: Modify the data of an existing student
   - **Retrieve**: Look up and display a student record
3. The data structure is updated accordingly
4. The result is displayed to the user

# Example

## Insert
```text
> Insert student
Name: Ahmed
ID: 1023
Grade: A
```

## Retrieve
```text
> Search by ID: 1023
Name: Ahmed | ID: 1023 | Grade: A
```

## Delete
```text
> Delete student ID: 1023
Student removed successfully
```
